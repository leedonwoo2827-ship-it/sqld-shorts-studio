# 한국어 발음 사전 (PRONUNCIATION)

## 무엇
TTS(Supertonic)가 영문 약자·외래어를 음절로 어색하게 읽는 걸(예: `MOOC → 엠오오씨`)
바로잡기 위해, **합성 직전에 텍스트를 치환**하는 사전.

## 어디에 / 어떻게 편집
- 파일: `config/pronunciation_map.yaml` (`표기: 읽는법` 매핑)
- 웹: 상단 **📖 발음 사전** → 추가/수정/삭제. 저장 즉시 반영(서버 재시작 불필요).

```yaml
rules:
  MOOC: 무크
  API: 에이피아이
  WWW: 더블유더블유더블유
```

## 언제 적용되나 (중요)
- **합성용 텍스트에만** 적용된다. **자막(SRT)에는 원문 그대로** 들어간다.
  → 화면 자막은 "MOOC", 읽는 소리는 "무크".
- 단어 추가 후 **그 씬만 다시 생성**하면 반영된다(전체 재생성 불필요).
- 자동 처리도 있음: 모르는 영문 약자는 글자단위 음역, 연도(`1989년→천구백팔십구년`),
  숫자+단위(`27분→이십칠분`) 변환.

## 동작 위치 (코드)
- `voicewright/pronunciation.py` (`PronunciationMap.apply`)
- 엔진이 합성 직전 `engine.synth` 에서 적용, 파일 변경 시 핫리로드.
